## Web APIs Demo

### Run Demo
```
git clone https://github.com/abraj/awa-demo.git
cd awa-demo
npm start
```

The above command starts a local demo server on [http://localhost:3000](http://localhost:3000/)

[Hosted Demo](https://awa-demo.web.app/)
